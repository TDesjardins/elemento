Elemento Samples

This site contains five different implementations of the TodoMVC web app using Elemento:

  • Builder - Using the builder API
  • Templated - Using HTML templates
  • Errai - Using Errai DI and the builder API
  • Dagger - Using Dagger DI and HTML templates
  • GIN - Using GIN DI and HTML templates

All samples use the same key to persist the todo items in the local storage. So you can switch between the samples and continue working on your tasks seamlessly ;-)
